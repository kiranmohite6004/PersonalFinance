# Personal Finance

## Project Features

- Track your income and expenses.
- Generate financial reports.
- Budgeting tools to help manage spending.
- User-friendly interface with graphical representations of data.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/kiranmohite6004/PersonalFinance.git
   cd PersonalFinance
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```

## Usage

- After starting the application, you can create an account or log in.
- From there, you can start adding your financial data, which can be categorized into different expenses and income sources.
- Explore the dashboard to view your financial status and trends.

## Structure

- **/src**: Contains the source code of the application.
- **/public**: Static files like index.html and images.
- **/tests**: Contains unit tests for the application.
